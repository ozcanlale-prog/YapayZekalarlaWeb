## YapayZekalarlaWeb

🌦️ AI-Driven Weather Dashboard: The Multi-Model BenchmarkBu proje, modern web geliştirme süreçlerinde farklı yapay zeka modellerinin ve kod asistanlarının (Claude Code, Gemini, Cursor, Anti-gravity) performansını, tasarım yaklaşımlarını ve kod mimarilerini karşılaştırmak amacıyla geliştirilmiştir.Her bir versiyon, aynı temel gereksinimler doğrultusunda (Open-Meteo API kullanımı, Geocoding entegrasyonu ve responsive tasarım) ilgili AI aracı tarafından sıfırdan oluşturulmuştur.📊 Karşılaştırma MatrisiAşağıdaki tablo, geliştirme sürecinde her bir AI modelinin ön plana çıkan karakteristik özelliklerini özetlemektedir:Geliştirici ModelOdak NoktasıUI FrameworkÖne Çıkan ÖzellikClaude CodeMimari & Temiz KodTailwind CSSHata yönetimi ve modüler JS yapısı.GeminiVisual StorytellingTailwind + PhosphorPremium Glassmorphism ve zengin veri görselleştirme.CursorGeliştirme HızıBootstrap/TailwindDosya yapısı ve hızlı iterasyon kabiliyeti.Anti-gravityİnovatif ÇözümlerCustom CSS/TailwindAlışılagelmişin dışında UI bileşenleri ve UX akışı.🛠️ Teknik MimariTüm modeller, aşağıdaki veri akış şemasını takip ederek kullanıcıya en güncel hava durumu verilerini sunar:Input: Kullanıcıdan şehir ismi alınır.Geocoding: Şehir ismi koordinatlara ($Latitude, Longitude$) dönüştürülür.Weather Fetch: Open-Meteo API üzerinden anlık, saatlik ve günlük veriler çekilir.UI Rendering: Gelen JSON verisi modellerin kendine has UI bileşenleri ile ekrana basılır.🚀 Öne Çıkan Versiyon: Gemini "Premium" EditionÖzellikle Gemini tarafından geliştirilen versiyon, kullanıcı deneyimini (UX) ön plana çıkaran şu özelliklerle donatılmıştır:Safe Index Mapping: Kullanıcının yerel saatiyle API'den dönen verileri senkronize eden özel algoritma.Dynamic Backgrounds: Hava durumuna göre (yağmurlu, güneşli vb.) değişen atmosferik arka planlar.Hourly Timeline: 24 saatlik veriyi kaydırılabilir bir "time-capsule" yapısında sunma.📂 Klasör YapısıBash├── 01-claude-code-version/    # Temiz mimari odaklı versiyon
